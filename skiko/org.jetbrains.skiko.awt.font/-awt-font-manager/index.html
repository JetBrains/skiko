AwtFontManager

class AwtFontManager

Manages available fonts, both at the system level and custom ones.

The main entry point is getTypefaceOrNull:

val fontManager = AwtFontManager()
val myTypeface = fontManager.getTypefaceOrNull("My Font")
Content copied to clipboard

While system fonts represent global state and are shared across all instances, custom fonts are only available on the instance(s) on which they have been registered.

In order to free up memory, you should remove custom fonts you don't need anymore.
