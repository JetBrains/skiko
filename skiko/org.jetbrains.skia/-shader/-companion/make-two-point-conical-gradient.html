makeTwoPointConicalGradient

fun makeTwoPointConicalGradient(start: Point, startRadius: Float, end: Point, endRadius: Float, gradient: Gradient, localMatrix: Matrix33? = null): Shader

Returns a shader that generates a conical gradient given two circles, or returns null if the inputs are invalid. The gradient interprets the two circles according to the following HTML spec. http://dev.w3.org/html5/2dcontext/#dom-context-2d-createradialgradient

Parameters

start

The center of the circle for this gradient

startRadius

Must be positive. The radius of the circle for this gradient

end

The center of the circle for this gradient

endRadius

Must be positive. The radius of the circle for this gradient

gradient

Description of the colors and interpolation method

localMatrix

Optional local matrix, may be null
